# Declaração de Missão
**Projeto:** dgp.cnqp_lib

---

# 1. Missão
"Fornecer uma biblioteca Python robusta e reutilizável para extrair dados estruturados do Espelho de Grupos de Pesquisa do CNPq, permitindo que sistemas institucionais enriqueçam suas bases de dados de forma ágil e confiável."

---

# 2. Visão
"Ser a biblioteca de referência para integração de dados do CNPq em pipelines ETL acadêmicos, garantindo extração estruturada e de fácil consumo."

---

# 3. Valores do Projeto
1.  **Reusabilidade**: Código modular e orientado a objetos para fácil integração.
2.  **Confiabilidade Técnica**: Extração robusta usando Playwright e parsing OO.
3.  **Simplicidade**: API clara (`python -m dgp_cnpq_lib <url>`) e saída JSON padronizada.
4.  **Manutenibilidade**: Testes automatizados e documentação atualizada.
